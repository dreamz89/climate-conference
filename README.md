# Climate Solutions Conference

## Deployment

Live: https://climate-conference.vercel.app/

## Techstack

Framework:  **Next.js (SSG)**  
CI/CD:       **Github Actions**  
Hosting:    **Vercel**  
Styling:    **Tailwind CSS**  
CMS: **Storyblok**
